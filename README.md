# PersonalScale
Project to build digital scale using ESP32, HX711 ADC and OLED-Display.

## Getting started
To start, you need the Arduino IDE with dependencies installed. Hardware setup is described here: ct.de/yj2t.

## Dependencies

- [HX711](https://github.com/bogde/HX711), [Basecamp](https://github.com/merlinschumacher/Basecamp), [SSD_13XX](https://github.com/sumotoy/SSD_13XX)
